# 🎯 InterviewPilot

An AI-powered interview preparation platform that analyzes your resume and generates personalized interview questions to help you practice technical and behavioral interviews with confidence.

> Built with **Next.js**, **TypeScript**, **PostgreSQL**, **Drizzle ORM**, **Google Gemini AI**, and **Better Auth**.

---

## 🚀 Live Demo

**Website:** https://interview-pilot-neon.vercel.app/

---

## ✨ Features

### 🔐 Authentication
- Secure user authentication with Better Auth
- Persistent user sessions
- Protected dashboard and interview routes

### 📄 Resume Management
- Upload resumes in PDF format
- Secure cloud storage using Cloudinary
- Resume history and management
- Delete or view uploaded resumes

### 🤖 AI-Powered Interview Generation
- Resume parsing using Google Gemini AI
- Personalized interview questions based on:
  - Skills
  - Projects
  - Experience
  - Technologies
- Dynamic interview generation for every uploaded resume

### 🎤 Interactive Interview Experience
- Question-by-question interview flow
- Voice input using the Web Speech API
- Manual text input support
- Timer for each question
- Real-time progress tracking

### 📊 AI Evaluation & Reports
- AI-generated answer evaluation
- Individual question feedback
- Overall interview score
- Detailed performance report
- Interview history for future review

### 🎨 Modern UI
- Responsive dashboard
- Clean and intuitive interface
- Mobile-friendly design
- Smooth user experience
